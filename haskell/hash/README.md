# hash

A experimental shell written in Haskell

# Features

I want to implement those functionalities

- [ ] Pipe support
- [ ] Redirect support
- [ ] Ordinary math expr
- [ ] Heredocument support
- [ ] Builtin commands
- [ ] POSIX capability

And more, I want to implement something from Haskell, like:

- [ ] Type for each variable
- [ ] Lambda function
- [ ] Monad(?)
