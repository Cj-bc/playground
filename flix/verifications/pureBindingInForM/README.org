* Pure binding in forM

As of flix 0.73.0, ~forM~ can contains pure local variable binding using ~VARNAME = EXPRESSION~ syntax.

#+begin_src flix

def f(): Result[Unit, Int32] =
  forM (
    _ <- Ok(1); // First line should be generator to suppress unrelated error
    const = (1)
  ) yield const

def main(): Unit \ IO = println(f())

#+end_src

This behaviour is confirmed with flix 0.71.0, 0.72.0, 0.73.0.
